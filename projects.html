discordrb

Discordrb is a gem for accessing the Discord API. I've been involved with the development since 2018 and now act as the project lead and primary developer.

I've made quite a few contributions; adding missing features, implementing new features, optimizations, and refactoring existing code. My largest accomplishments with the project so far are improving the websocket efficiency by 300%, updating the voice sending functionality, and updating to support multiple user activities.

Working as the maintainer has given me valuable experience in leading an OSS community, working with git, and futhering my knowledge of RESTful APIs. It has also given me a very valuable opportunity to socialize with Discord employees and get insight to the design process of a large company.

vox-etf

This was made to enable the use of erlang's ETF format inside of Ruby via a native extension written in C++. It makes use of Discord's erlpack, as well as custom serde methods, to interact with Discord's gateway. It was originally made for my own API wrapper, vox, but will also work for anyone that wishes to use it to create their own custom gateway solution.

doc-o-tron

This is the documentation bot for the discordrb channel on the unofficial Discord API server. It's features are as follows:

  • Search and format YARD documentation
  • Evaluate Ruby, Crystal, and C via a sandboxed eval service
  • Reference github issues/pull requests from the discordrb repository
  • Show the source of a given method


I'm especially proud of this bot, mostly because of how the YARD => Discord formatting works. I wrote custom ERB templates for the YARD generator to use, so it outputs a message formatted specifically for Discord using markdown and ASCII art. The end result is somewhat buggy as I encounter new issues where certain tags don't fit what my template expects, but I enjoy it none the less.

eviler-winrm

This was originally a fork of Hackplayers/evil-winrm because their code was frankly offensively bad. However, I soon realized that it would take much less effort to completely remake their tool from the ground up with a more modular and configurable approach. Things like a configurable and remote rendered prompt, various log modes, tab completion of plugin commands as well as remote directory completion, and enhanced color support are just a small number of reasons why eventually our entire penetration testing team was using eviler-winrm over the more popular tool fairly quickly.

caldera

Music bots are a large and important part of the Discord bot community. One of the most popular tools used by these music bots is Lavalink, allowing the voice connection logic to be outside of their application and allow for playing of different media sources such as soundcloud or youtube. This library interfaces with a lavalink server in a generic way that allows discordrb or vox to play music entirely externally for the first time since discordrb's inception.

copyup

This is a little tool written in Rust that allows a user to quickly download a LICENSE file via its SPDX identifier (e.g: MIT, 0BSD, MPL-1.0, etc). I made it so I would have a reason to use Rust and so I could be lazy about downloading licenses for new projects.
